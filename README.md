# Hospital Management System

Goal : To Build a Hospital Management System Website that provides role-based access and functionalities for Admins, Doctors and Patients.

## Overview
The Hospital Management System (HMS) is a complete solution designed to manage hospital operations efficiently. It allows for smooth management of patients, doctors, appointments, billing, and reports, ensuring better healthcare service delivery.

## Features
- Patient Management: Register, update, and track patient information.
- Doctor Management: Manage doctor profiles, specialization, and schedules.
- Appointment Scheduling: Book, update, and cancel patient appointments.
- Reports & Analytics: Generate reports for patients, doctors, and appointments.

## Technologies Used
- Programming Language: Python 
- Database: SQLite
- Frameworks: Flask 
- Frontend: HTML, CSS
